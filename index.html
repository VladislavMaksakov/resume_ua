У процесі розробки додатку, я використовував мову програмування C# та середовище Visual Studio. Проект включав створення інтерфейсу користувача, де користувач міг вибрати файли для стиснення. Для стиснення аудіо та медіа файлів, я використовував пакети NuGet, такі як "NAudio" та системні бібліотеки. Ці пакети надавали функціональність для обробки аудіо та відео даних, включаючи стиснення.
